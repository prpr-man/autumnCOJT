# Asobeat

新しい"アソビ"が始まる．

# Description

心拍センサとGPSによる位置情報を取得して従来の"アソビ"に新しいカルチャーを提供します．
無限のアソビを手に入れよう．

# Tech

Backend: Node.js + Express + Mongoose 

* https://nodejs.org/en/
* http://expressjs.com
* http://mongoosejs.com

Client: iOS(Swift)

* http://www.apple.com/jp/swift/
